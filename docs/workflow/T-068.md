**Статус:** DONE

**Агент:** GARMIN_YEAR_RECAP

## Описание

Создан полный годовой отчёт по спортивным активностям 2025 года на основе данных Garmin Connect через MCP инструменты.

## Выполненные действия

1. **Сбор данных** — активирован MCP Garmin и получены все активности за 2025 год
2. **Анализ статистики** — использован готовый файл [data/garmin/training.json](../../data/garmin/training.json) с годовыми итогами
3. **Генерация отчёта** — создан JSON-файл [data/garmin/year-recap-2025.json](../../data/garmin/year-recap-2025.json) с детализированной статистикой
4. **Оформление поста** — создан красивый пост на русском языке [docs/coaching/2025-year-recap.md](2025-year-recap.md)
